Welcome to the Archive!

This is a collaborative archive. You can help build it by adding folders and files.

To add a file, please follow these steps:

  1. Navigate to the correct Degree folder (e.g., BSc Physics).
  2. Click the 'Add New Batch' button to create a folder for your batch (e.g., 2021-2025).
  3. Enter the new batch folder and create a Year folder.
  4. Enter the year folder and select the correct Semester.
  5. Once inside the semester folder, click the 'Add New File' button to upload your papers or notes with a proper caption.
